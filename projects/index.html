Projects made by RL

All of the projects created by RL have been detailed below, in rough chronological order. For some projects you can click on the gold title to be brought to an external page. For mods that have multiple versions, the version number may be displayed under Tags. Click here if you want to view the game descriptions on the old site. Further information about the rom hacks are usually available at the download page.

"But how do I get roms?" This website can't give you roms but if you're ever bored with not being able to play rom hacks you can check out stuff like this github repository to brush up on your website skills or take a look at RL's favorites on the Internet Archive for interesting books to read.

All projects are available for download here: Internet Archive

Title Date Tags Description
Notegames 2013-2014 none A collection of games created on notebook paper.
Powergames 2014 none A collection of games created in Apache OpenOffice Impress which mainly use the mouse as controls. One can imagine the physics engine is not very beefy. These are the very first computer games created by RL.
Kaizo Man 2016 kaizo, megaman, mod, NES, v1.1 A kaizo mod for Mega Man that was tested and completed only using savestates. It's RL's oldest hack and was created using level editing utilities for the game. The game was not made with human limits in mind. Apply patch to Mega Man (U) for NES.
MEGA MAN Dance Machine 2016 one-off, megaman, mod, NES Fun low effort mod that makes Mega Man run when he normally stands still and stop when he normally runs. It was made in like 20 minutes. Apply patch to Mega Man (U) for NES.
Mega Man Jelly Doughnut Version 2016 one-off, megaman, mod, NES Movement mod loosely based on Rockman 4 Burst Chaser. Mega Man has a higher jump and a faster movement speed enabling you to breeze through levels much faster. Speedruns of the mod are generally fun as well since that's what was in mind when the mod was created. Nothing else was changed other than movement. Apply patch to Mega Man (U) for NES.
Kaizo Man 2 2016 kaizo, megaman, mod, NES Rockman 2 but kaizo. Same idea as the prequel, Kaizo Man. No ASM was used to create this so it's just a nice level rearrangement that's also not made with human capabilities in mind. Arguably less brutal than the original Kaizo Man. Apply patch to Rockman 2 (J) for Famicom.
Super Mario Kart Battle & Chase 2017 mod A mod of Super Mario Kart with level design roughly based off of Mega Man Battle and Chase. It's rather fun if not a little bit unfaithful to the source material, as it still contains Mario Kart front to back. All of the tracks in the game have been redone including Battle Mode, so it can be considered a full game. Characters have also been arbitrarily rebalanced. Apply patch to Super Mario Kart (U) for Super Nintendo.
Balanced Melee 6/26/2017 smash, mod, GCN A mod that doesn't take itself very seriously, to the point where RL assumed the emphasis in "Balanced" would be implied as a joke. It's an unbalanced mod akin to Brawl-. Movement and combo game are the main things that have been increased, as well as recovery. Pretty fun to play with friends every now and then. Patch includes the 20xx hackpack at the download page but original character files are available at the gold link. Apply patch to Smash Bros Melee v1.02.
Competitive Melee 2/24/2018 smash, mod, GCN A mod that purely aims to rebalance the game by doing small adjustments to keep characters very in line with their original intent while making them more playable. Few special moves have been drastically changed, but the problem areas of the worse characters have been noticeably changed. A little bit out of date, as some of the higher characters were also changed a bit, when the goal was to only change the weak characters. Give it a try. Patch includes the 20xx hackpack at the download page but original character files are available at the gold link. Apply patch to Smash Bros Melee v1.02.
Project Meme 2018 smash, mod, Wii, ongoing A wonderfully crafted mod of Project M which adds new moves to every character, changes how moves function, and overall create a fun and hilarious mod. The custom moves and the near universal increase in the number of jumps makes for a free flow experience. Over time there has been many updates and balance patches but the game has always been fun. It's one of the top mods on gamebanana, and it's highly worth playing. The installation guide is available at the link for the download.
WNLFC + MFTOP 2018 mod, DS, ongoing The full title for this hack is "Webslinger's New Lava Fish Castle + Mega Fan's Tower of Pain", and yes, there will be an update that adds even more letters, some completely unnecessary. The mod is originally based off a handful of Mario Maker levels that RL and Manmanboy created. There are 26 levels between the two games, spanning Worlds 1-4. MFTOP is in world 3-4 and can be accessed via the pipe at the start of the first level. Every level has 3 star coins, and some parts of the map may be blocked off because world map editing wasn't on the radar at that point in time. The difficulty of this hack is higher than a traditional rom hack but it's more focused on challenge and precision rather than things being unfair/cheap. Some changes to the levels have even been made to make them more fair, something that RL doesn't often do once a hack is released. Apply to New Super Mario Bros. (USA) for Nintendo DS.
Stamina Melee RG 6/19/2020 smash, mod, GCN The melee mod where RL never got to show what the 'RG' was supposed to mean. It notoriously was patched incorrectly when first released so it was a monumental pain to get running, as many users ran into an error in xdelta, which has now been addressed and fixed in a recent update. Anyway, it's a mod for Smash Bros Melee where the system and stagelist have been reworked to HEAVILY favor Stamina mode as the main game mode. By some stroke of luck, it's been supported by top melee players Essy, Polish, and Khryke (see reference). Apply to Smash Bros Melee v1.02.
JMF Training 8/12/2020 JMF, mod, NES, v1.05 A deprecated training mod for Joy Mech Fight that has been eclipsed by ShenScript and JMF CE. Auto unlocks characters. This was the tournament standard used sometimes if vanilla wasn't preferred. Later versions were notably buggy and had incomplete features. Using certain versions in ShenScript may be preferred but it's honestly better to switch over to CE at this point. Apply patch to Joy Mech Fight (J) for Famicom.
CDLtranslator 2021 program, Java A Java program that translates the FCEUX Code/Data logger (.cdl) file data into something more readable. Released as source code so throw it in whatever IDE you like.
Rockman Universe's Tournament 5/21/2021 homebrew, Wii, v0.5 RL's first attempt at creating an entirely new fighting game, which was written in C and most everything was done from scratch. RenDev is credited wherever this project is released as he helped RL greatly in getting the project to where it was aimed. This fighting game is inspired by the Mega Man Power Battle/Fighters games for the arcade, which were multiplayer but sadly not 1v1. No future updates are currently planned, however you can experience the game's past releases. No source code released as of yet. You can run this game by loading it in Dolphin or putting the folder in the apps folder of your SD card to play on a homebrewed Wii. Supports Gamecube controllers and Wii remotes.
JMF Retranslated 8/21/2021 JMF, one-off, mod, NES A retranslation mod that CE was built off of. Originally built from the 12/15/2000 translation mod by AlanMidas. Some translations were poorly done so the move and character names were double verified. CE comes included with this mod, once again. Apply patch to Joy Mech Fight (T-eng) by AlanMidas.
refreshinglemona.de 9/1/2021 website, HTML This website. If you wanna know more go to the About page.
ShenScript 9/13/2021 lua, JMF, script A lua training script for Joy Mech Fight that is named after Shenlong, the coolest character in the game. Supports basic training functions such as frame advantage, hitbox display and more. Overall a very useful tool for understanding specifics about the game. Should support any original or modded versions of Joy Mech Fight. Use with Bizhawk emulator.
JMF Community Edition 4/10/2022 ASM, JMF, mod, NES Commonly referred to as CE, this mod is tournament standard for Joy Mech Fight as of 2022. Most changes in this mod can be modified in some way to allow for tournament ruleset flexability. 5 minute timer, P2 change CSS and Leo removed, menu cursor auto hovers on P1 vs P2, anti-epilepsy patch for the intro cutscene, and Houou had a special removed. Move and character names have also been changed to be more accurate to the original Japanese names while still tying in the English-speaking community. Apply patch to Joy Mech Fight (J) for Famicom.
San Guo Zhi Improvement 9/15/2022 ASM, bootleg, mod, NES Asder/NTDEC's San Guo Zhi (Three Kingdoms) for the NES with the added ability to play as the boss characters in a regular match. For P1 you can select them on the CSS normally, for P2 hold A or B when loading into a match after selecting any character. Low effort ASM injection. Apply patch to San Guo Zhi - Qun Xiong Zheng Ba (Asia) (Ja) (Unl) for Famicom.
Super Blaster VII Turbo 28 Control Fix 7/21/2023 ASM, bootleg, mod, NES One of the better Cony Soft games which unfortunately dropped 70% of inputs in the original game. Fixed it. Low effort ASM injection, once again. Apply patch to Super Blaster VII Turbo 28 (Unl) for Famicom.
RL vs SLK Randomizer Tool 4/18/2024 program, python A halfway-decent tool to randomly generate Pokemon teams created as a companion to the several Pokemon randomizer runs that RL and SLK did. Useful for randomized movesets that you'd want to mess around with.
